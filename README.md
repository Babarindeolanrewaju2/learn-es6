#### learn-es6

Learn es6 while converting your code.

![demo gif](https://raw.githubusercontent.com/siddharthkp/learn-es6/master/demo.gif)

#### Install

`npm install -g learn-es6`

#### Usage

Run `es6` in the repository you want to learn in
